Overview

An AI-powered Loan Approval and Credit Risk Assessment System built using Machine Learning, Streamlit, and Google Gemini API. The application evaluates applicant financial data, predicts loan approval, assesses credit risk, generates AI-driven financial recommendations, and creates downloadable PDF reports.

Features
Loan Approval Prediction using Machine Learning
Credit Risk Assessment
AI Financial Advisor (Gemini API)
Approval Confidence Score Analysis
Personalized Financial Recommendations
Approval Confidence Score Analysis
Personalized Financial Recommendations
PDF Report Generation
Interactive Streamlit Dashboard
Technologies Used
Python
Streamlit
Scikit-learn
Pandas
NumPy
Google Gemini API
FPDF
Project Workflow
User enters financial details.
ML model predicts loan approval.
Risk level and approval score are calculated.
Gemini AI generates financial advice.
PDF assessment report is generated.
Results are displayed through a Streamlit interface.
